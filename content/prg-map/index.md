## `$AFFB` - ランレングス圧縮BGデータを PPU へ同期的に転送するルーチン {#addr-AFFB}

### 引数

| 引数 | 内容               |
| --   | --                 |
| A    | データアドレス下位 |
| Y    | データアドレス上位 |

### データ形式

| 型   | 内容         |
| --   | --           |
| u16  | PPU アドレス |
| u8[] | 圧縮データ   |

圧縮データは以下のいずれかの要素の配列:

* `0xFF`: データ終端
* `0..=0x7F`: 無圧縮データ
* `0x80..=0xFE` `<data>`: `<data>` を指定個数繰り返す。個数は `(先頭Byte)-0x80`。
